# ──────────────────────────────────────────────────────────────────────────
# robots.txt — 2026 "block training, allow retrieval" template
#
# robots.txt is the real lever that controls AI visibility (llms.txt is ignored).
# Strategy: ALLOW retrieval/search bots that drive citations + referral traffic;
# DECIDE per-policy on training bots. Customize before shipping.
#
# Gotchas:
#  1. Blocking OAI-SearchBot removes you from ChatGPT search answers.
#  2. Deprecated strings (Claude-Web, anthropic-ai) no longer work — don't rely on them.
#  3. ~27% of sites ALSO accidentally block AI bots at the CDN/Cloudflare/WAF layer — audit that.
#  4. noindex blocks AI citation too — a page must be indexable to be cited.
#  5. Never block your sitemap path.
#  6. Non-compliant bots (Bytespider, stealth crawlers) ignore robots.txt — enforce at WAF/server.
# ──────────────────────────────────────────────────────────────────────────
